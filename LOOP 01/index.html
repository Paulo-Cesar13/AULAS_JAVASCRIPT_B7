WHILE

O segnificado de while é enquanto, basicamente, ele vai repetir um comando enquanto um parametro não for atendido!!

EXEMPLO

var contador = 0;

while (contador < 10) {
document.write(contador);
contador++;
}

Nesse exemplo acima, declaramos uma variavel de valor 0, e criamos um contador, o código, ai elabarado vai acrecentar um enquanto o valor for menor que 10!!

EXEMPLO 02

var contador = 50;

while (contador > 10) {
document.write(contador);
contador--;
}

Nesse exemplo acima, declaramos uma variavel de valor 50, e criamos um contador, o código, ai elabarado vai diminuir um enquanto o valor for maior que 10!!

EXEMPLO 03

var contador = 10;

while (contador !== 0) {
document.write(contador);
contador--;
}

Nesse exemplo acima, declaramos uma variavel de valor 10, e criamos um contador, o código, ai elabarado vai diminuir um enquanto o valor for diferente de 0!!

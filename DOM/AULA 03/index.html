Selecione com Queryselector!

Ele substitui os 4 comandos aprendidos anteriormente, com ele podemos alterar os elementos pelo Id, class, tag ou name.

Exemplos:

document.querySelector('#exemplo') - Altera pelo Id.
document.querySelector('.exemplo') - Altera pela classe.
document.querySelector('div') - Altera pela tag.
document.querySelector('name') - Altera pelo Name.

Obs: Quando houver mais de um elemento, no que ele retorna arry, ele mostrata apenas o primeiro item!
para mostro todos os itens do elemento, deve-se usar o seguinte comando.

document.querySelectorAll('.exemplo')

Exemplo:

documente.querySelector('#exemplo').innerHTML = 'Alteração';

Obs: Você identifica os elemento da mesma forma como faz no css, se for Id - # (jogo da velha)
Se for class - . (ponto)

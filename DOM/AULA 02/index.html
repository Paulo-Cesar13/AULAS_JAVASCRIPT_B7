Selecionando Elementos ( 4 funções)

Existe 4 principais comando para selecionar objetos na tela.

1 - Selecionando os objtos pelo ID:

document.getElementById('Nome do ID'), Nessa primeira parte do codigo, é para encontra o id, que queremos modificar ou manipular.

document.getElementById('Nome do ID').innerHTML, Esse segundo parte mostro onde e como vai ser mudado o objeto, nesse caso vai ser mudado dentro do html, Obs: inner quer dizer dentro. Podemos ainda modificar ou acrencentar um codigo html.

Exemplo:

Neste primeiro exemplo vamos modificar o nome Paulo, para o nome César, utilisando o codigo: document.getEçementById('Nome do ID').innerHTML = "César";

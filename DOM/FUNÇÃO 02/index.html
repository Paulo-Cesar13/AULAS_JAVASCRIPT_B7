COMO PASSAR PARAMETROS

function alterar(parametro) {
document.getElementById("titulo").innerHTML = parametro;
document.getElementById("campo").value = paramentro;
}

alterar("Tudo o que for escrito aqui vai substituir a palavra paramentro");

A função acima é a mesmo da aula 01, com a diferença que agora foi declarado um paramentro para ela. Nos ainda podemos declara mais de um parametro para mesma função, os parametros funcionam como variaveis.
